# 🐍 SNEK

A neon arcade Snake game built with pure HTML, CSS, and JavaScript — no libraries, no frameworks, just vibes.

## 🎮 Gameplay

- Control the snake using **WASD** or **Arrow Keys**
- Swipe to control on mobile
- Eat food to grow and score points
- Don't hit the walls or yourself!

## 🍔 Food System

| Food | Rarity | Points |
|------|--------|--------|
| 🍔 Burger | Rare (30%) | +10 |
| 🧋 Milktea | Common (70%) | +5 |

## ✨ Features

- Smooth interpolated snake movement
- Dual food types with dynamic spawn probability
- Speed increases as you eat more
- Particle burst on every eat
- Screen flash effects
- Web Audio API sound effects (no external libraries)
- Live scoreboard with burger & milktea counters
- Dynamic "Best Food" tracker
- Mobile swipe support
- Game Over + Restart screen

## 🛠️ Built With

- HTML5 Canvas
- Vanilla JavaScript (modular, clean)
- Web Audio API
- CSS animations & custom properties

## 🚀 How to Run

Just open `snake-game.html` in any browser. That's it.
